# Architecture Documentation

This folder contains the implementation-aligned architecture material for the collaborative editor AI system.

Subfolders:

- `diagrams/` -> C4 diagrams, ER diagram exports, and editable Mermaid sources
- `adr/` -> Architecture Decision Records

Use the Mermaid files in `docs/diagrams/` when the report or system behavior changes so the editable source stays in sync with the PNG exports.
